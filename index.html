Coding Journey

My passion for coding begin after reading an article about how the author taught himself how to code through an online course and figured I would test it out and see if I liked coding myself.

I started off with HTML CSS and Javascript and I was hooked. Shortly afterwards I signed up for a coding bootcamp at George Washington University where I learned front and backend development. Some of the languages we covered were mongoDB, SQL, jQuery, reactJS, Express, NodeJS and many more.

I love designing websites making them more user friendly and appealing. I tend to spend a lot of time working with the frontend and its where I am most comfortable but don't mind getting dirty with some gritty backend development on projects that I am passionate about.

Currently I am learning PHP and using WordPress as well as strengthing my skills in MERN development.
